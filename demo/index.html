    Welcome to Estuary Live Demo
    Hello!

    Welcome to the Estuary design framework live demo! This project is still a work in progress and need's contributors like you to help improve it! You can find the source code for Estuary framework on GitHub published open source for everyone to use freely at NO COST - tho donations are more than appreciated!

    This design framework intends to re-create the default Kodi theme. It was originally created for the project Bodi - a project that intends to re-create Kodi with more features for developers & more. (a work in progress). Tho it can also serve as a great design framework for your next desktop / mobile app, be a usefel resource to help skin your favorite exisiting media client / server etc.. or just be useful to help create a looking website in a unique HTML / CSS / JS design framework.

    You can also find a more modern looking version of Estuary branded as "Estuary Redux" - here. (also a work in progress)


    Estuary was created by Jared Van Valkengoed from https://github.com/MarketingPipeline
